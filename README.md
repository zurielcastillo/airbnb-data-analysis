# Airbnb Mexico City Data Analysis

## Project Overview

This project presents an end-to-end data analysis of Airbnb listings in Mexico City. The workflow includes data cleaning, preprocessing, exploratory data analysis (EDA), and business insights using Python.

The project is organized following common data science practices, separating raw data, processed data, notebooks, reusable source code, and reports.

## Objectives

- Clean and preprocess the Airbnb listings dataset.
- Explore pricing patterns across neighborhoods and room types.
- Identify trends through exploratory data analysis.
- Generate business insights supported by visualizations.
- Demonstrate reproducible data analysis workflows.
